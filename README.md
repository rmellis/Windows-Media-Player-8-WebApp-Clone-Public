# Windows-Media-Player-8-WebApp-Clone-Public
A nostalgic web recreation of Windows Media Player 8 Series. Features local audio/video playback, YouTube &amp; HLS radio streams, 10-band EQ, spatial audio, beat-synced visualizations, in-browser MKV remuxing via FFmpeg.wasm, voice control, recording, and 12+ custom skins. happily but occasionally angrily built with vanilla JS, HTML, and CSS!.
